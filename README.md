Program ini dibuat untuk mempermudah pengelolaan data buku di perpustakaan, dengan menerapkan konsep Doubly Linked List untuk manajemen data di memori secara dinamis dan efisien.



Fitur/menu: 

1. Tambah Buku (Insert Last)
2. Tampilkan Semua (Traversal)
3. Cari Buku (by Kode/Judul)
4. Sorting (Ascending/Desc)
5. Edit Data (Update by Kode)
6. Hapus Buku (Head, Tail, \& Tengah)
7. Pinjam Buku (Update Stok \& Status)
8. Kembalikan Buku (Restock Otomatis)
9. Simpan File (dataPerpustakaan.txt dengan Delimiter)
10. Keluar (check last change)



Logic Dasar:

* Struktur Data: Menggunakan struct dengan pointer kiri dan kanan.
* Sistem File: Data disimpan permanen ke dalam file .txt. Saat program dibuka, data otomatis di-load kembali ke RAM (Linked List).
* Kelebihan: Mendukung judul/nama dengan spasi dan pengurutan data yang fleksibel.
